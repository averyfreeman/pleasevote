# Project Refactor Rules
- **Stack:** Transitioning to Tailwind CSS, TypeScript, and Node 20.
- **Style:** Use functional React components and standard ES modules.  
- **Patterns:** Combine concerns or improve other patterns for easier understanding. 
- **Workflow:** Always create a new branch for each task; do not merge to main automatically.
- **Environment:** Use secrets defined in the Jules UI for any API calls during testing.
